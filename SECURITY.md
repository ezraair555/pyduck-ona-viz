# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.1   | :white_check_mark: |
| 0.1.0   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities **privately** by emailing the
maintainer at [ezraair555@gmail.com](mailto:ezraair555@gmail.com). Do not
open a public issue for security bugs.

Include as much detail as possible:

- Affected version(s)
- Steps to reproduce
- Impact assessment
- Any suggested remediation

The maintainer will acknowledge receipt within 48 hours and provide an initial
assessment within 5 business days. Fixes will be released as a new patch
version along with an advisory in `CHANGELOG.md`.
