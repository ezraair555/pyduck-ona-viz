# API overview

`pyduck-ona-viz` exposes ten public visualization entry points plus the
shared `theme` helpers.

| Module | Public functions |
|---|---|
| [`theme`](theme.md) | Palette, typography, figure factory, and style helpers. |
| [`span_control`](span_control.md) | `span_of_control`, `span_vs_depth` |
| [`org_chart`](org_chart.md) | `org_chart_tree`, `reporting_chain_walk` |
| [`network_viz`](network_viz.md) | `centrality_dashboard`, `silo_map` |
| [`hr_viz`](hr_viz.md) | `attrition_heatmap`, `compensation_equity` |
| [`hierarchy_viz`](hierarchy_viz.md) | `hierarchy_depth_heatmap` |
| [`dashboard`](dashboard.md) | `summary_dashboard` |
